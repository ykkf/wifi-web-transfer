# Web版 Wi-Fi File Transfer

インストール不要でブラウザから直接利用できるファイル転送Webアプリです。Node.js + Expressで構築されており、ローカル実行とVercel（サーバーレス）へのデプロイの双方に対応しています。

## 特徴
- アップロード後に一意の共有URLとQRコードを発行
- 個別ファイルダウンロード ＆ 一括ZIPダウンロード
- モダンでレスポンシブなダークテーマUI
- 自動で端末の状況に合わせて最適な通信方法を選択（Base64 JSON転送）

## 【重要】Vercel(無料版)での制約事項
Vercelの無料枠（Serverless Functions）で動かす場合、Vercelの仕様により**一度に送信できるファイル合計サイズの上限は約 4.5MB** となります。大容量の動画や高画質な写真を転送する場合は、**ローカルPCで `start_app.bat` を使って起動してください（ローカルの場合は容量無制限です）**。

また、今回はMVP（データベース不要のメモリ管理）のため、Vercel上では無アクセス時間が続くとセッションが消滅する可能性があります。

## 使い方1: 自分のPCをサーバーにする（特におすすめ・無制限）

Windowsパソコンをお使いの場合は、同梱のバッチファイルを使うだけで一発で起動できます。

1. **`start_app.bat` の実行**
   パソコン内にある `start_app.bat` を右クリックし、「管理者として実行」をクリックします（※ファイアウォールを開放するため）。
   
2. **自動セットアップ**
   初回の Node.js パッケージのインストールから、サーバーの起動、ブラウザの立ち上げまで全て自動で行われます。

3. **iPadやスマホからのアクセス**
   黒い画面（コマンドプロンプト）に表示された `iPad / phone: http://172.16.x.x:8080` などのURLを、iPadのSafariなどに入力すれば繋がります。

## 使い方2: Vercelへのデプロイ（インターネット経由）

インターネット上に公開したい場合は、GitHub経由でVercelに簡単にデプロイできます。

1. **GitHubへ公開**
   GitHub Desktopを使ってこのフォルダをコミットし、リポジトリを作成＆Publishします。

2. **Vercelとの連携**
   Vercelのサイトから「Add New Project」を選び、作成したGitHubリポジトリをインポートするだけで自動で公開されます。
   ※URLが分割されてメモリが失われないよう、`vercel.json`で全てのルーティングを単一のエントリポイントに統合しています。

## 機能の仕様
- **フロントエンド**: Vanilla JS, `qrcode.js` を活用したSPAライクな動作。ファイルをBase64へエンコードする仕様で、Vercel環境との互換性を確保。
- **バックエンド**: Node `express`, `uuid`, `archiver` (ZIP生成) を使用。
